27. v-if 和 v-show 的相同点和不同点分别是什么？（5分）

相同：控制元素显示和隐藏
不同：v-if控制节点渲染，v-show控制节点样式display：none属性


28.v-if 和 v-show 应用场景分别是什么？（4分）
v-if:节点不频繁显示/隐藏切换
v-show:节点频繁显示/隐藏切换

29.Vue 无法检测到 data 中数据的哪些变化？如何解决？（6分）
对象属性添加或删除、数组下标、长度 检测不到

解决：$set

30.Vue 中如何进行父子组件之间的传值？（6分）

父组件-》子组件：props
子组件-》父组件：自定义事件


31.computed 和 methods 的区别是什么？（4分）
computed  是属性，直接通过属性名使用,自带缓存
methods  是方法，需要()调用，没有缓存

32.组件中的 data 为什么必须是函数？（5分）
如果data是对象不是函数，那么在复用同一个组件时data的值会受到影响(地址是同一个)
而只有data是函数时，每次切换或者其他操作在复用组件时都会重新调用函数，而data就是一个新的地址，不会影响以前的data
