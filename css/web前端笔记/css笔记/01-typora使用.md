# typora使用

## 什么是typora工具？

​	typora其实是一种以md为后缀，支持实时预览的 Markdown 文本编辑器。主要用于开发方向记录笔记，优势在于可以快速创建代码块。md是文件格式。

简称为：**文本编辑器。**
