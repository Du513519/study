# 数据库

##  概念

数据，英文叫做database,简称DB,就是一个可以存放数据的仓库，我们可以对里面的数据进行CRUD操作

## 为什么要专门来学数据库

一般的文本、world只能处理少量的数据,并且不方便查询

而我们学习的数据库能够处理海量数据，拥有独特查询方式

## 分类

### 关系型数据库 (表和表是具有关系的)

~~~
mysql (免费)    sqlserver    oracle 
~~~

### 非关系型数据库 (表和表是互不相干的)

~~~
mongoDB (json格式)   redis ....
~~~



## 结构

~~~
数据库
   集合(表)
   	   文档(数据)
~~~
